# Life Routes Dashboard

## Files
- index.html

## Run locally
Double-click `index.html` or open it in your browser.

## Upload to GitHub

1. Create a new repository on GitHub.
2. Name it (example: `life-routes-dashboard`).
3. Upload `index.html` to the repository.
4. Commit the changes.

## Enable GitHub Pages

1. Open **Settings**.
2. Select **Pages**.
3. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
4. Save.

Within a minute or two your site will be available at:

https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/

## Customize

You can:
- Change default percentages.
- Add target savings amounts.
- Add localStorage to save progress.
- Integrate with Fort Hope Trading Post.
